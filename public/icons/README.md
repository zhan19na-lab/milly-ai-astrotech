# Icon Placeholders

Place the following files here before deploying:

| File                | Size    | Usage                              |
|---------------------|---------|------------------------------------|
| icon-192x192.png    | 192×192 | Android home screen, PWA splash    |
| icon-512x512.png    | 512×512 | Android splash, maskable icon      |

## Quick generation (requires ImageMagick or Squoosh)

Use any 1024×1024 master logo and export to the two sizes above.
Theme color: #6D28D9 (Vivid Violet)
Background:  #0D0D11 (Space Black)
